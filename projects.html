OngakuVault

OngakuVault is a web application designed to facilitate audio archiving. It is designed to run on Windows, Linux, and Docker. OngakuVault allow users to download audio files from multiple websites, store their metadata and make changes to selected fields.

PictureColorDiffusion

My attempt at automating the colorization of 2D pictures using Stable Diffusion, relying on the automatic111 web UI, its API, and ControlNet. A link to an AI YOLO Segmentation model I trained for tests is also available.
